# Tappetite — Brand Kit

Всё, что нужно, чтобы вести Instagram и делать контент **в стиле Tappetite**.

Tappetite — игра-турнир на вкус: тапаешь любимое из пары (X или Y), в финале
получаешь свой топ-3 и текстовый «типаж». Эстетика — журнальная: кремовый фон,
serif-заголовки, тёплый терракотовый акцент.

## Что внутри
| Папка | Что | Как использовать |
|---|---|---|
| `logo/` | Логотип-«t», вордмарк «tappetite», иконка, OG-картинка | Ставь на аватар/посты. Не растягивай, не перекрашивай. |
| `colors/` | `palette.png` + `palette.txt` — все hex | Копируй цвета в Canva/Figma/Reels. |
| `fonts/` | `FONTS.md` — какие шрифты и где скачать бесплатно | Установи их себе, используй в постах. |
| `screenshots/` | Живые экраны игры (home / дуэль / результат) | Для reels, каруселей, сторис. |
| `templates/` | `ig_post_square.html` — готовый редактируемый пост | Открой в браузере, поменяй текст, сделай скриншот. |
| `BRAND_GUIDE.md` | Цвета, шрифты, логотип, голос бренда | Прочитай один раз — поймёшь «как можно, как нельзя». |
| `INSTAGRAM_PLAYBOOK.md` | Идеи постов, форматы, расписание + **готовые промты для ИИ** | Твоя рабочая шпаргалка. |

## Быстрый старт (5 минут)
1. Открой `BRAND_GUIDE.md` — запомни 3 цвета и 3 шрифта.
2. Открой `INSTAGRAM_PLAYBOOK.md` — там 5 типов постов и готовые промты, которые
   можно вставлять в свой Claude, чтобы генерить тексты и картинки в нашем стиле.
3. Для первой картинки — открой `templates/ig_post_square.html` в браузере,
   поменяй два варианта («Кофе» / «Чай»), сделай скриншот 1:1. Готов первый пост.

Ссылка на игру для шапки профиля: **isverifiedby.me** (и Google Play — Tappetite).
