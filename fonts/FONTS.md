# Шрифты Tappetite

Все три — **бесплатные** (можно ставить и использовать в постах). Скачай чистые
версии с Google Fonts и установи себе на компьютер, чтобы использовать в Canva /
Figma / PowerPoint / где угодно.

| Шрифт | Роль | Скачать |
|---|---|---|
| **Bodoni Moda** | Логотип + крупные serif-заголовки. Элегантный, тонкий. Курсив — для акцента. | https://fonts.google.com/specimen/Bodoni+Moda |
| **Onest** | Обычный текст, подписи, UI. Чистый sans-serif. | https://fonts.google.com/specimen/Onest |
| **JetBrains Mono** | Мелкие подписи КАПСОМ ВРАЗРЯДКУ («ТВОЙ ТИПАЖ», «РАУНД 1 ИЗ 5»). | https://fonts.google.com/specimen/JetBrains+Mono |

## Как использовать вместе (иерархия)
1. **Заголовок** — Bodoni Moda, крупно (можно курсив на второй строке).
2. **Подпись-капс** — JetBrains Mono, мелко, ЗАГЛАВНЫМИ, с разрядкой между букв.
3. **Текст** — Onest, обычный размер.

В Canva: найди эти шрифты по названию (Bodoni Moda / Onest / JetBrains Mono —
все есть в Canva бесплатно), не подбирай «похожие».

> В templates/ уже всё зашито — HTML-шаблон сам подгружает эти шрифты.
